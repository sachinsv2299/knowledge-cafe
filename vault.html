IITH VAULT

Network Attached Storage (NAS) Guideline

This guide explains how IITH members can securely access and use the IITH VAULT (NAS) using their LDAP credentials. The NAS provides centralized file storage, synchronization, backup, and secure file sharing for authorized users.

IITH VAULT Ecosystem

Seamlessly connect via Web browser, Desktop Agent, or Mobile app.

2

First-Time Login (Access Method: Web)

  1. Open the Web Portal

    Navigate to https://vault.comp.iith.ac.in/ from any modern browser.

  2. Enter Credentials

    Enter your IITH LDAP username (do not enter your full email address) and LDAP password.

    IITH-Vault Sign In
    Username (e.g., cs21btech11001)
    Password

  3. Set Up 2-Factor Authentication

    To protect your account, you will be prompted to set up a second verification step. Click Start and select Verification code (OTP).

  4. Install Authenticator App & Verify

    Download the Synology Secure SignIn mobile application (available on the App Store and Google Play Store) or a compatible third-party TOTP authenticator (such as Google Authenticator or Microsoft Authenticator). Scan the provided QR code and enter the 6-digit verification code generated by the Synology Secure SignIn, Google Authenticator, or Microsoft Authenticator app to complete the setup.

  5. Configure Backup Email

    Enter your IITH email address as a backup. This ensures you can receive emergency codes if you lose your mobile device. Click Send Verification Mail, verify the setup, and click Done.

  6. Access Synology Drive

    After successful login, click the Synology Drive icon from the application dashboard.

  7. Explore the Interface

    Familiarize yourself with available sections: My Files, Shared Items, and Profile/Settings.

3

Synology Drive Client (Access Method: Desktop App)

A. Installation

Install Synology Drive Client from the official Synology download page:
https://www.synology.com/en-eu/dsm/feature/drive

Click “Download Client” button and choose the appropriate version for your operating system:

Windows (exe/msi) Mac (dmg) Ubuntu (deb)

B. Configuration

1. Connect to Server

Open the application and enter the connection details:

  • NAS: vault.comp.iith.ac.in
  • Username: LDAP Username
  • Password: LDAP Password

Click Next to proceed.

2. Choose Task Type

Sync Task

Two-way real-time sync

Backup Task

One-way backup

Select Sync Task for standard usage.

3. Set Up Folders

Select the NAS folder to synchronize (usually /home) and choose a local directory on your computer (e.g., C:\Users\Name\SynologyDrive).

Click Next through the remaining setup windows, then click Done and Open Folder.

4

Re-login (Existing)

  • Use the saved server profile if available in the client.
  • Re-enter your LDAP password if it has recently changed.
  • Restart the Synology Drive Client if you encounter persistent connection issues.
5

File Management

  • Drag and drop files to upload quickly.
  • Create folders using a proper, consistent naming format.
  • Use version-aware sync where enabled to recover old edits.
  • Avoid storing sensitive data outside of approved, secure folders.
6

File Sharing & Permissions

Easily collaborate by securely sharing files and folders with specific users.

How to Share:

Web: Right-click file/folder → Share

Desktop (Windows): Right-click → Show More Options → Synology Drive → Get Link

  • Permissions (Internal): Share securely via Private invitees or grant Internal view/edit access to anyone with an IITH account.
  • Public Link (External): Create a public link to share files with anyone. Always set a Password and Link Expiration date to protect sensitive data.
  • Always revoke shares when collaboration is complete.

Storage Quota

To check your storage limit and current usage:

  1. Open account/profile settings in the web portal or desktop client (Global Settings).
  2. Navigate to the Usage tab.
  3. Review storage usage against your remaining quota.
/homes 80.5 GB of 500.0 GB

Tip: Clean obsolete files when nearing limits.

8

Security Best Practices

  • Avoid accessing the Vault from shared computers.
  • Do not share your LDAP credentials with anyone.
  • Use strong endpoint security on all synced devices.
  • Report any suspicious access or activity immediately through the Computer Centre Support Portal.
9

Troubleshooting & Support

Invalid Login

Verify your LDAP password is correct and active.

Sync Failures

Check network connectivity and ensure you have sufficient local disk space.

Permission Denied

Confirm your share access or group permissions with the administrator.

Browser Issues

Clear your browser cache, disable conflicting extensions, or try incognito mode.

Still facing issues?

If you encounter any issues not covered here, please raise a support ticket through the Computer Centre Support Portal.
